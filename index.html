Executive Summary

Masalah Utama: Ekosistem loyalitas Topremit saat ini bersifat transaksional satu arah. Pengguna setia tidak mendapat apresiasi khusus, dan TR Coins dengan jumlah kecil menjadi "frozen assets" karena minimum penarikan Rp10.000.

Solusi: Topremit Elite - program loyalitas bertingkat (Explorer ‚Üí Pro ‚Üí Elite) dengan Smart Redeem yang memungkinkan koin digunakan tanpa minimum.

Impact: Retention Rate +15% (60% ‚Üí 75%), Frekuensi Transaksi 2x lipat, Customer Lifetime Value +25%.

Speaker Notes: Value proposition utama adalah mengubah TR Coins dari "saldo tambahan" menjadi "alat retensi strategis" melalui tier system dan smart redeem.

Current Context

Ekosistem Saat Ini

  • TR Coins: Cashback poin per transaksi
  • Referral: Bonus Rp120.000 (3x Rp40.000)
  • Affiliate: Komisi untuk partner

Critical Issues

  • Minimum withdrawal Rp10.000 ‚Üí frozen assets
  • User mudah switch karena selisih Rp5.000
  • Tidak ada pembeda user baru vs loyal
User Transaksi ‚Üí Dapat Koin ‚Üí Withdraw ke Bank ‚Üí No Engagement Loop
Speaker Notes: Pasar remitansi Indonesia sangat kompetitif. Tanpa program loyalitas yang kuat, Topremit rentan kehilangan user ke kompetitor.
